# Estándares de Codificación

## Introducción

Este documento define los estándares de codificación utilizados en el Sistema de Gestión de Supermercado desarrollado en Java.

## Objetivos

- Mantener orden en el código
- Facilitar mantenimiento
- Mejorar legibilidad

## Nombramiento de Variables

Se usa camelCase

Ejemplo:

precioProducto

## Clases
